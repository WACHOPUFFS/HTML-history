HTML 1.0

The original version of HTML was HTML 1.0. It had very limited features which greatly limited what you could do in designing your web pages.

HTML 2.0

HTML 2.0 then arrived and included all the features of HTML 1.0 plus several new features for web page design. Until January, 1997, HTML 2.0 was the standard in web page design.
Advantages

  • HTML 2.0 did not support tables.

  • The document structure was simplified as much as possible to speed up editing.

    • HTML 3.2

    HTML 3.2 version was published by W3C in early 1997. It became an official standard for any browser till January 1997. It included many new and improved abilities for HTML, and promised far more powerful opportunities for webmasters to design their pages. Today it is practically supported by most of the browsers.
    Advantages

  • the latest advances in web page development.

  • Incorporated text that fkiws around the images.

  • This version was capable of creating tables and providing support for extra options for form elements.

  • It can also support a web page with complex mathematical equations.

    • HTML 4.01

    HTML 4.01 version was released on December 1999, and it is a very stable version of HTML language. This version is the current official standard.
    Advantages

  • Has CSS or cascading style sheets to add visual effects to web pages

  • Was a more hassle-free version of HTML.

  • Added buttons and iFrame to be able to have sites inside other site

    • HTML5

    HTML5 is the newest version of Hyper Text Markup language. The first draft of this version was announced in January 2008. There are two major organizations one is W3C (World Wide Web Consortium), and another one is WHATWG (Web Hypertext Application Technology Working Group) which are involved in the development of HTML 5 version, and still, it is under development.
    Advantages

  • HTML 5 incorporates new elements not covered in HTML 4.01.

  • There are new tags.

  • The possibility of introducing audio and video directly into the web without the need for plugins or browser add-ons is introduced.

  • New controls for forms that were previously only possible with JavaScript or CSS.

  • Simplified doctype tag.
